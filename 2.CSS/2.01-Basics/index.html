2.1 - CSS basics

Ways of writing CSS:

  1. Inline: as an attribute in the opening tag of an element.
  2. Internal: in the same html files inside the style tag in the head.
  3. External: in a seperate CSS file, and then linking that file inside html head.

Basic stbtax of CSS:

Types of selectors (a way to target a specific element):

  1. Tag: h1,p,div
  2. Class: .class
  3. Id: #id
